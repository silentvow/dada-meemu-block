# DaDa & Meemu's s Treasure Hunt

Play: https://dada-meemu-block.vercel.app/

This is a fan-made, second-generation adaptation of a brick-breaking game by React and Pixi.js.

## Development

```bash
$ npm install
$ npm run dev
```

## Build

```bash
$ npm run build
```
